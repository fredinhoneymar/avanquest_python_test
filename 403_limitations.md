# Historical Data Access Limitations

This document outlines the limitations of accessing historical data using the Football Data API v4 (https://api.football-data.org/v4/).   
JSON responses are provided unedited for auditing purposes.

You can also use the app to fetch full data for a specific endpoint e.g. '/competitions/PL/matches' (it will be stored in the cache folder),  
and then look for the available historical information e.g. year '2020'.

I have been in contact with Daniel from Football-Data.org regarding the access limitations.  
Email correspondence is included at the end of this document.

- The API returns a 200 OK response when trying to access the competitions resource.
- The API key used has the necessary permissions to access this endpoint.
- The response includes the subscription plan information: TIER_ONE (free tier), which comes with limitations.

```bash
curl -XGET 'https://api.football-data.org/v4/competitions' -H "X-Auth-Token:12abfbaacdab48bc8948ed6061925e1f"
```
